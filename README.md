=
O QUE É ESTE PROJETO?
=

Uma aplicação web para explorar os Açores que permite:
- Ver locais turísticos das 9 ilhas
- Consultar meteorologia em tempo real
- Criar itinerários personalizados
- Avaliar e comentar locais
- Guardar favoritos
- Exportar itinerários para PDF

=
ARQUITETURA
=

5 APIs desenvolvidas em MuleSoft:

1. Experience API - Interface com o frontend
   URL: https://experienceapi-turismometeo-13pr64.5sc6y6-3.usa-e2.cloudhub.io/api

2. Process API - Orquestração de processos
   URL: https://processapi-turismometeo-13pr64.5sc6y6-1.usa-e2.cloudhub.io/api

3. Database API - Acesso à base de dados MySQL
   URL: https://system-api-database-13pr64.5sc6y6-2.usa-e2.cloudhub.io/api

4. OpenMeteo API - Integração meteorologia
   URL: https://system-api-openmeteo-13pr64.5sc6y6-2.usa-e2.cloudhub.io/api

5. Geoapify API - Integração routing/mapas
   URL: https://system-api-geoapify-13pr64.5sc6y6-1.usa-e2.cloudhub.io/api
